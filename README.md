> To start and run this project

- bundle install
- rails s

To get a local copy up and running, follow these steps.

- git clone https://github.com/NatanimA/PaymentsIQ.git

### Prerequisites

In order to run this project you don't need any prerequisites:
```
ruby
```

```
postgres
```
